# lamihani-homepage

라미한의원 홈페이지 — 기존 아임웹 페이지(`https://lamihani.imweb.me/`)를 정적 사이트로 복제한 버전입니다.

## 구성

| 파일 | 설명 |
| --- | --- |
| `index.html` | 원페이지 메인 (진료과목 · 치료법 · 의료진소개 · 진료시간 · 위치안내 · 공지사항) |
| `css/style.css` | 전체 스타일 (반응형 포함) |
| `js/main.js` | 모바일 메뉴 · 스크롤 등장 애니메이션 · 맨 위로 버튼 |

## 특징

- 순수 HTML/CSS/JS, 빌드 도구 불필요 — 파일을 그대로 열거나 정적 호스팅에 올리면 동작
- 아임웹 브랜드 색상(오렌지 `#ff7727` / `#ff9252`)과 섹션 구성 유지
- 데스크탑/모바일 반응형
- 이미지는 기존 아임웹 CDN(`cdn.imweb.me`)을 참조 — 추후 자체 이미지로 교체 가능

## 로컬에서 보기

```bash
python3 -m http.server 8000
# http://localhost:8000 접속
```

## 참고

- 예약/상담 버튼은 네이버 지도 예약, 카카오톡 채널로 연결됩니다.
- 공지사항은 정적 페이지 특성상 게시판 기능 없이 안내 영역만 표시됩니다.
